Allgemeines

Diese digitalSTROM Applikation erhöht den Komfort und senkt den Energieverbrauch von Personen, die alleine in einer Wohnung leben.

Sobald in einem Raum das Licht eingeschaltet wird schaltet die Applikation das Licht im vorhergenden Raum aus. So brennt das Licht jeweils nur in einem Raum.

Für jeden Raum können eigene Einstallungen gemacht werden:

  • Keine Aktion: Dieser Raum wird nicht beachtet.
  • Ausschalten: Das Licht wird sofort ausgeschaltet, wenn in einem anderen Raum das Licht eingeschaltet wird.
  • Langsam Ausschalten: Das Licht während einer Minute gedimmt, wenn in einem anderen Raum das Licht eingeschaltet wird.
